#  Foodly - Трекер Питания

Веб-приложение для отслеживания калорий и питательных веществ.

##  Возможности

- Поиск по базе 30,000+ продуктов
- Добавление приёмов пищи с подсчётом КБЖУ
- Статистика за день/неделю/месяц
- Навигация по датам
- Группировка по типам приёма (завтрак/обед/ужин)

##  Технологии

- PHP 8.3
- SQLite
- Vanilla JavaScript
- CSS3

##  Установка
```bash
# Клонировать репозиторий
git clone https://github.com/dmitridemenkov/foodly.git

# Импортировать базу продуктов
python import_to_sqlite.py

# Запустить через DDEV или встроенный сервер PHP
php -S localhost:8000 -t public
```
